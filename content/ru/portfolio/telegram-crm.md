## Идея

Многие малые бизнесы не хотят учить менеджеров работать с CRM. Telegram знают все. Решение — CRM прямо в боте.

## Архитектура

- **Laravel** на сервере — хранит клиентов, сделки, историю
- **Telegram Bot** — единственный интерфейс для менеджеров
- **FSM (конечный автомат)** для навигации по меню бота
- **Webhook** + очереди для надёжной обработки сообщений

## Возможности

- Добавление и поиск клиентов
- Ведение воронки продаж (статусы сделок)
- История всех взаимодействий с клиентом
- Напоминания о follow-up
- Еженедельные отчёты боту в группу менеджеров

## Стек

Laravel 11 · PHP 8.3 · MySQL · Redis (очереди) · Telegram Bot API
