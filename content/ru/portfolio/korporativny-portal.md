## О проекте

Разработка корпоративного портала для производственной компании с 200+ сотрудниками. Система заменила хаотичный документооборот в WhatsApp и Excel.

## Технологии

- **Backend**: Laravel 11, PHP 8.3
- **Frontend**: Blade + Alpine.js + Tailwind CSS
- **База данных**: MySQL 8.0
- **Файловое хранилище**: MinIO (S3-совместимое)

## Функциональность

- Управление задачами с дедлайнами и исполнителями
- Согласование документов с цепочкой подписей
- Внутренние уведомления (web + Telegram)
- Система прав доступа (роли и разрешения)
- Дашборд с ключевыми метриками для руководителей

## Результат

Время на согласование документов сократилось с 3 дней до 4 часов. Все задачи и их статусы стали прозрачны для всей команды.
